What is the goal of this website?

The goal of this website is to use the Wikimedia REST API to highlight particularly popular Wikipedia articles and display some interesting data about them

Which API did you use?

  • Get a text extract & thumb summary of a page
  • Get the most viewed articles for a project
  • Get pageview counts for a page

How do I use this website?

For Top Articles tab, you can search the top english or french articles of a date under the format of YYYY/MM/DD. You can also bookmark and unbookmark them.

For Saved Articles tab, it contains all the saved articles. You can unbookmark them from as well.

For Activity Chart, you can search a specific article and it will return the monthly views for the past twelve months. As for the input, please write it under this format: "Jackie Chan" and not "Jackie_Chan". If you are unable to find the article, please go to Wikipedia, find the article and copy the title and paste it in the textbox.
